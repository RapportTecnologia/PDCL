## PDCL - Plan, Do, Check Logs

### Overview

Este repositório tem como objetivo armazenar informações, artigos, estudos, exemplos e provas de uso da nova metodologia de desenvolvimento **PDCL - Plan, Do, Check Logs**.

### O que é PDCL?

PDCL é uma metodologia de desenvolvimento que estende o ciclo clássico PDCA (Plan, Do, Check, Act) com foco específico em **Logs** para análise e melhoria contínua de processos de desenvolvimento de software.

- **Plan**: Planejamento das atividades e definição de métricas
- **Do**: Execução das tarefas planejadas
- **Check**: Verificação e análise dos resultados através de logs
- **Logs**: Registro sistemático para rastreamento e auditoria

```

|-- studies/       # Estudos de caso
```
